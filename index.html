  • JavaScript (JS)

    JavaScript is a scripting or programming language that allows you to implement complex features on web pages — every time a web page does more than just sit there and display static information for you to look at — displaying timely content updates, interactive maps, animated 2D/3D graphics, scrolling video jukeboxes, etc. — you can bet that JavaScript is probably involved. It is the third layer of the layer cake of standard web technologies, two of which (HTML and CSS) we have covered in much more detail in other parts of the Learning Area.

  • Operators

    An operator performs some operation on single or multiple operands (data value) and produces a result.

  • Variables

    JavaScript variables are containers for storing data values.

  • Variable naming conventions

    You should not use any of the JavaScript reserved keywords as a variable name. These keywords are mentioned in the next section. JavaScript variable names should not start with a numeral (0-9). They must begin with a letter or an underscore character. JavaScript variable names are case-sensitive.

  • Functions

    A function is a block of organized, reusable code that is used to perform a single, related action.

  • Methods

    JavaScript methods are actions that can be performed on objects. A JavaScript method is a property containing a function definition.

  • Arguments

    Function arguments are the real values passed to (and received by) the function.

  • Parameters

    A parameter or a formal argument, is a special kind of variable, used in a subroutine to refer to one of the pieces of data provided as input to the subroutine.

  • Return

    A return statement causes execution to leave the current subroutine and resume at the point in the code immediately after the instruction which called the subroutine, known as its return address.
